# HOW TO INSTALL ZPHISHER

* `apt update`
* ` apt upgrade -y`
* `pkg install git curl wget php -y`
* `git clone https://github.com/NIHALHACKER/zphisher`
* `cd zphisher`
* `bash zphisher.sh`
